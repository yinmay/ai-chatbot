# RenderMe - Project Description

## 项目概述

RenderMe 是一个基于 AI 的智能职业助手，专门为前端开发者提供简历优化和技术面试准备服务。通过先进的 AI 技术和专业的面试经验，帮助求职者提升简历质量和面试表现。

## 核心功能

### 1. 简历优化 Agent

**功能特点：**
- 专业的简历评审和优化建议
- AI 驱动的技能评分系统（5-10分制）
- STAR 方法指导项目经历描述
- 量化的改进建议和优化方向
- 支持生成定制化简历模板

**工作流程：**
1. 用户粘贴简历内容
2. AI 自动分析简历结构和内容
3. 调用技能评分工具进行量化评估
4. 生成详细的优化报告，包括：
   - 整体评分（0-100分）
   - 教育背景优化建议
   - 专业技能评分和改进方向
   - 工作经历优化要点
   - 项目经验提升方案
   - 具体的优化前后对比示例

**技术亮点：**
- **evaluateSkills Tool**: 根据毕业年份和技能列表，智能评估技能的深度和广度
- **自动化评分**: 基于工作年限、技能数量、技术栈现代性等多维度评分
- **个性化建议**: 针对不同经验级别提供定制化的优化建议

### 2. 模拟面试 Agent

**功能特点：**
- 真实的前端技术面试模拟
- 8-10 个问题覆盖完整面试流程
- 实时反馈和专业点评
- 全面的面试表现报告
- STAR 方法行为面试指导

**面试流程：**
1. **自我介绍和动机** (1-2题)
   - 自我介绍评估
   - 求职动机了解
   - 离职原因考察

2. **编程基础题** (1题)
   - JavaScript 语言特性
   - 代码规范和质量

3. **算法题** (1题)
   - 初中级难度
   - 思维和实现能力

4. **场景设计题** (1题)
   - 技术方案设计
   - 系统架构思维

5. **项目经验深挖** (3-4题)
   - 项目介绍
   - 技术挑战
   - 性能优化
   - 成就亮点

6. **反向提问和综合点评**
   - 候选人提问环节
   - 整体表现评估
   - 优势和不足分析
   - 改进建议

**评估标准：**
- ✅ 加分项：清晰的表达、具体的案例、量化的数据、STAR 方法应用
- ❌ 减分项：表述混乱、缺乏例证、说前公司坏话

### 3. 智能意图分类

**功能：**
- 自动识别用户意图（简历优化、模拟面试、技术问答）
- 智能路由到对应的专业 Agent
- 无缝的用户体验

**支持的意图类型：**
- `resume_opt`: 简历优化相关
- `mock_interview`: 模拟面试相关
- `related_topics`: 技术问答和职业咨询

## 技术架构

### AI Agents 架构

```typescript
lib/ai/agent/
├── classify.ts          # 意图分类系统
├── resume-opt.ts        # 简历优化 Agent
├── mock-interview.ts    # 模拟面试 Agent
├── common.ts            # 共享工具函数
└── index.ts             # Agent 路由中心
```

**设计特点：**
- 模块化设计，每个 Agent 独立负责特定功能
- 统一的消息处理和数据库持久化
- 支持工具（Tools）扩展

### AI Tools

```typescript
lib/ai/tools/
├── resume-template.ts   # 简历模板生成器
├── evaluateSkills.ts    # 技能评分工具（集成在 resume-opt.ts）
├── create-document.ts   # 文档创建
├── update-document.ts   # 文档更新
└── ...
```

**resumeTemplate Tool:**
- 支持多种技术栈（React, Vue, Full-stack, Node.js）
- 适配不同经验级别（初级、中级、高级）
- 根据工作年限动态生成技能列表
- 提供完整的简历结构模板

**evaluateSkills Tool:**
- 输入：毕业年份、技能列表
- 输出：评分（5-10）、工作年限、期望技能数、总结、建议
- 评估维度：
  - 技能数量（基于工作年限）
  - 技能深度描述
  - 避免弱词汇（"了解"）
  - 技术栈现代性

### 技术栈

**前端框架：**
- Next.js 16 (App Router)
- React 19
- TypeScript
- Tailwind CSS + shadcn/ui

**AI 集成：**
- Vercel AI SDK 6.0
- 支持多模型提供商：
  - Anthropic (Claude 4.5 系列)
  - OpenAI (GPT-4.1, GPT-5.2)
  - Google (Gemini 2.5, Gemini 3)
  - DeepSeek (Chat, Reasoner)

**数据持久化：**
- Neon Serverless Postgres
- Drizzle ORM
- Vercel Blob Storage

**认证：**
- Auth.js (NextAuth v5)
- 支持多种登录方式

## 核心特性

### 1. 专业的 System Prompts

**简历优化 Prompt:**
- 明确的专业领域定义
- 详细的评审要点
- 具体的优化建议和示例
- 工具使用指南

**模拟面试 Prompt:**
- 结构化的面试流程
- 明确的提问原则
- 详细的评分标准
- 综合点评模板

### 2. 智能评分系统

**技能评分算法：**
```typescript
基础分: 5
+ 技能数量评估: 0-2分
+ 技能深度评估: 1-2分
+ 描述质量评估: 0-0.5分
+ 技术栈现代性: 0-0.5分
= 总分: 5-10分
```

**期望技能数量计算：**
```typescript
expectedSkills = min(15, 5 + yearsOfExperience * 2)
```

### 3. 用户体验优化

**建议输出格式：**
- 使用 Markdown 格式化
- 清晰的章节划分
- ✅/❌ 标记加分减分项
- 具体的优化前后对比

**实时反馈：**
- 流式响应，即时展示
- 进度可见，体验流畅

## 使用场景

### 场景 1: 简历优化

**用户故事：**
> 作为一名前端开发者，我想优化我的简历，让它更有竞争力。

**使用流程：**
1. 用户说："帮我看看简历"
2. 系统识别意图，路由到简历优化 Agent
3. Agent 提示用户粘贴简历内容
4. 用户粘贴简历
5. Agent 调用 evaluateSkills 工具评分
6. 生成详细的优化报告

### 场景 2: 模拟面试

**用户故事：**
> 作为求职者，我想通过模拟面试提升面试表现。

**使用流程：**
1. 用户说："我想进行面试模拟"
2. 系统识别意图，路由到模拟面试 Agent
3. Agent 了解用户背景（技术栈、经验、目标职级）
4. 开始面试流程，依次提问
5. 对每个回答给予简短点评
6. 完成 8-10 个问题后，给出综合评价

### 场景 3: 生成简历模板

**用户故事：**
> 作为求职者，我想快速生成一份专业的简历模板。

**使用流程：**
1. 用户说："帮我生成一份 React 工程师的简历模板"
2. 系统调用 resumeTemplate 工具
3. 用户提供参数：技术栈、工作年限、目标职级
4. 生成定制化的简历模板
5. 用户根据模板填写个人信息
